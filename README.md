# Personal Landing Page

A clean, minimal personal landing page built with Tailwind CSS and Vanilla JS.

🌐 **[Live demo →](https://ginogiorgi.github.io/asyncLanding/)**

## Stack

- HTML + Vanilla JS
- [Tailwind CSS](https://tailwindcss.com/) via CDN

## Run locally

Open `src/index.html` directly in a browser, or serve with any static server.
